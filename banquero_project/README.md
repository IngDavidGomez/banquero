# Algoritmo del Banquero - Flask

Proyecto web educativo para demostrar el algoritmo del banquero con un solo tipo de recurso.

## Estructura

```text
banquero_project/
├── app.py
├── banquero.py
├── requirements.txt
├── README.md
├── run.bat
├── static/
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── main.js
└── templates/
    └── index.html
```

## Ejecución en Windows

1. Abre CMD en la carpeta del proyecto.
2. Instala Flask:

```bash
pip install -r requirements.txt
```

3. Ejecuta la app:

```bash
python app.py
```

4. Abre en el navegador:

```text
http://127.0.0.1:5000
```

También puedes ejecutar `run.bat`.
